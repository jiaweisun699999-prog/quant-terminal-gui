## 新人从这里开始

1. 克隆本仓库。
2. 打开 **`Quant-Terminal/README.md`**，按其中的 **「快速部署」** 安装依赖并运行或打包 Windows exe。

若虚拟环境建在仓库根目录（与 `Quant-Terminal` 并列的 `.venv`），进入子目录后可用：

```powershell
cd Quant-Terminal
..\.venv\Scripts\Activate.ps1
```

或直接在 `Quant-Terminal` 下创建 `.venv`（推荐，与《快速部署》一致）。
